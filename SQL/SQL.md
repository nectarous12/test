# SQL

SELECT 컬럼, 계산값

FROM 테이블 명

WHERE 조건

GROUP

BY 그룹화

HAVING 그룹화에서 사용되는 조건



### EX

SELECT 상품번호 FROM DB명.PRODUCT;

SELECT customernumber FROM classicmodels.custmers;

### 문법

SUM() : 집계함수 

`* : 모든결과 조회

AS : 컬럼명 변경

DISTINCT : 중복 제거

FROM :  정보 호출

WHERE : 정보 추가

BETWEEN : 시작점~끝점 데이터
